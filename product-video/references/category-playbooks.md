## 共同自然主义

默认目标是可信的电商实拍，不是概念广告：

- 使用普通住宅、教室、小店操作台等真实地点；不要默认豪宅、黑棚或空无一物的样板间。
- 使用一个明确时间段和一个主光源。优先白天窗光、普通吸顶灯或店内工作灯，不加体积光、烟雾和戏剧性光束。
- 机位保持眼平或手部操作高度，35–50mm 等效视角；近景只为看清接触，微距全片最多一次。
- 最多一名成年人或一组合理的手；人物只完成与商品有关的普通动作，不表演惊讶、欢呼或夸张反应。
- 场景保留 2–3 个有因果的生活痕迹，如桌上作业本、窗边椅子、杯封机旁的订单杯；不制造脏乱，也不把背景清成 CGI 空间。
- 每段只使用一个主体动作和一个有动机的相机行为。锁定镜头是有效选择，不要为了“动态”强行环绕或连续推拉。
- 变化要缓慢、可追踪并服从重力、接触、张力、摩擦、透明度和反射。结果通过真实动作出现，不用光效代替。
- 最后一帧回到同一件生活小事：继续工作、休息、开窗、收拾书包、递出饮料或摆回日常用品。不要跳成底座英雄海报。

默认排除：黑色摄影棚、展示底座、漂浮零件、粒子、火花、烟雾、发光护盾、能量波、热力射线、极端光斑、风暴式布料、无来源的水花、全镜面对称、无人豪宅、夸张前后对比、重复商品、额外手指、错误接触影、文字和图案变形。

## 参考图角色包

从同一 SKU 或同一图案/颜色款的相邻目录建立角色包。先视觉核对身份，再允许引用：

证据优先级：明确修改要求/实拍 SKU 图 → 原始商品图 → 结构细节图 → 详情页 → 主图合成 → AI 场景图。文件夹名称、翻译副本和历史版本不代表独立 SKU；低优先级合成图不能推翻高优先级结构约束。

| 角色 | 用途 | 最低要求 |
|---|---|---|
| `identity_truth` | 锁定款式、颜色、部件和包装 | 必须有；选最清晰正面或 SKU 图 |
| `scale_context` | 锁定与窗框、杯子、手指、桌面的比例 | 有则使用 |
| `installation_truth` | 锁定真实安装顺序、工具和接触面 | 安装类优先寻找 |
| `action_truth` | 锁定手、机器或商品实际运动 | 有则使用 |
| `result_truth` | 授权可见结果或功能主张 | 没有就不说结果功效 |
| `exact_print_truth` | 锁定姓名、字样、图案重复和方向 | 标签/贴纸/印刷膜必须寻找 |
| `world_plate` | 锁定房间、窗框、家具和主光 | 01 合格后可把 01 作为后续永久环境参考 |

禁止跨 SKU 拼接。文件名相近但颜色、图案、边框、卷芯、尺寸或配件不一致时视为不同商品。模糊文字保持模糊；只有 `exact_print_truth` 清晰可读时才允许复现文字。

## 六锚点日常闭环

所有路线共用生活节奏，不共用具体动作：

1. `need`：普通生活正在进行，出现一个小需求或未完成状态。
2. `ready`：商品和必要工具自然进入动作范围。
3. `aligned`：测量、清洁、对齐、放置或预备已经完成。
4. `applied`：关键安装、贴合、拉合、封合或摆放动作完成。
5. `verified`：用一个克制的可见动作检查结果；没有结果证据时只检查外观、边缘或位置。
6. `resumed`：手离开，人物回到原来的生活/工作，商品留在真实使用位置。

五段岗位固定为 `daily_trigger → prepare → apply → verify → return`。段落可以新增 `task_progress` 或 `context`，不必硬塞五个卖点。

常见扩展归属：磁吸门帘、浴帘、蚊帐和汽车遮阳卷帘通常进入 `window_textile`；沥水垫、地垫、餐垫和厨房/家具保护层进入 `surface_covering`；营业时间、门铃、调味/洗护标识进入 `adhesive_decal` 或需要精确小字时进入 `small_label`；魔术贴、磁条、压条、配件包和独立硬质家居品进入 `fallback_household`。汽车安装只在车辆静止时进行，不能遮挡驾驶员视线或后视镜。

## window_textile

适用：窗帘、门帘、卷帘、窗纱、磁吸纱网等。`form_factor` 使用 `textile_panel`、`tensioned_mesh` 或 `roll_to_sheet`。

推荐触发：准备午休、书桌反光、开窗通风、旧窗框等待装网、进出门口。推荐动作：`reach`、`measure`、`align`、`draw`、`hang`、`press_frame`、`roll`、`settle`、`open_close`、`resume_use`。

物理锁：

- 窗帘受重力垂落，褶皱从轨道/挂点向下延伸；拉动后有轻微惯性并自然停下。
- 纱网展开时可弯曲，安装完成后在边框间保持张力；不能变成玻璃、金属板或波浪旗帜。
- 卷帘/卷材的卷芯、展开方向和剩余卷径必须连续。
- 轨道、挂点、魔术贴、磁条或边框结构只能采用参考图可见方案。

可信闭环示例：人在书桌边感到窗边光线影响阅读 → 手伸向帘边 → 握住并沿轨道缓慢拉动 → 帘布闭合并自然垂落 → 桌面亮度变化被稳定看见 → 人继续阅读。只有来源证明遮光结果时才写“遮光”；否则写“帘布闭合、桌面光线变得更均匀”。

禁忌：飓风鼓动、瞬间完全无光、布料发光、虫群撞网、未经证明的隔音/降温/防蚊结论。

## window_film

适用：遮阳膜、隔热膜、磨砂膜、隐私膜、静电窗膜、装饰窗膜。`form_factor` 使用 `roll_to_sheet`。

推荐触发：玻璃待清洁、日常贴膜、更换旧膜、需要卷起/放下可卷式遮挡。推荐动作：`clean`、`measure`、`peel`、`spray`、`align`、`squeegee`、`trim`、`press_edge`、`roll_unroll`、`inspect`、`resume_use`。

物理锁：

- 薄膜始终是柔性薄片；未贴部分受重力轻垂，贴合区与玻璃共面。
- 需要现场裁切的整张膜在裁边前必须明显大于玻璃净尺寸，四边都连续超出；裁边后只有沿窗框分离的窄余料离开，成品边缘贴近玻璃边界。
- 先由同款说明确认静电湿贴、背胶湿贴还是其他固定方式；衬纸、喷液和工具只采用该 SKU 的真实流程，禁止把静电膜和背胶膜的步骤混在一起。
- 衬纸剥离边界、玻璃湿润区、刮板行进方向、气泡/水痕消退路径保持连续。
- 水痕、气泡和边缘余液只能随着刮板/等待逐段减少，不能在跨帧时突然全部消失；喷壶、刮板、布和余料使用后应留在合理位置或有明确收起动作。
- 玻璃仍保留受环境控制的反射和透视；磨砂/图案只改变参考图允许的区域。
- 单向隐私只在白天室外明显比室内明亮时表达：室外视角以同一片街景/天空反射为主，室内细节不易辨认；室内视角仍可看见窗外。不得暗示夜间开灯后仍有同等隐私。
- 可卷式产品遵循真实步骤：清洁玻璃 → 贴固定件 → 修边 → 安装 → 压边 → 卷起/放下。

可信闭环示例：普通卧室窗户待安装 → 擦净玻璃 → 沿边对齐膜片 → 刮板由中心向边缘缓慢排水贴合 → 手检查平整边缘 → 房间恢复正常使用。白天单向隐私需要外侧验证和内侧回看时，用同一扇门/同一窗框建立空间关系；字幕只说生活需求、光照条件和可见结果，不解说刮水或裁边。

禁忌：发光防护罩、紫外线/热量射线、冷热颜色对撞、瞬间消失的气泡、膜像硬板悬空、无证据隔热百分比或隐私承诺。

## adhesive_decal

适用：玻璃贴、窗贴、墙贴、标识贴、装饰贴。`form_factor` 使用 `decal_transfer`。

推荐触发：空白玻璃/墙面准备装饰或标识。推荐动作：`clean`、`select`、`peel`、`align`、`press`、`smooth`、`inspect`、`resume_use`。

物理锁：图案比例、方向、颜色块、留白和边缘轮廓不变；背纸与贴纸分离清楚；贴合从一个边缘或中心逐步发生；已贴区域与表面共面，不浮空、不拉伸、不多出第二张。

可信闭环示例：擦净玻璃 → 从背纸揭起一角 → 以窗框为基准对齐 → 手指/刮片逐段压平 → 退半步检查位置 → 继续原来的房间活动。

禁忌：图案自动飞到玻璃上、印刷重新绘制、贴纸变成立体发光物、墙面无接触阴影、凭空增加节庆道具。

## small_label

适用：姓名贴、小标签、文具贴、衣物/杯瓶识别贴。`form_factor` 使用 `decal_transfer`。

推荐触发：相似水杯、作业本或文具容易混淆。推荐动作：`select`、`peel`、`align`、`press`、`smooth`、`pack`、`retrieve`、`resume_use`。

物理锁：`exact_print_truth` 是最高优先级；姓名、字母、图标、边框和颜色不可改写。标签与手指、瓶身和本子的比例稳定；弧面贴合时边缘顺着曲率，不变成硬片。

可信闭环示例：桌上有两个相似水杯 → 选中对应姓名贴 → 揭开并对齐瓶身 → 拇指从中间向边缘压平 → 放入书包 → 之后按姓名快速取回。没有清晰文字真值时只展示图案色块，不生成可读新姓名。

禁忌：随机姓名、错别字、字体漂移、标签忽大忽小、爆炸式泼水、无证据“永久不掉”或“绝对防水”。

## sealing_film

适用：杯口封膜、封装膜、热封膜卷材。`form_factor` 使用 `consumable_film`。

推荐触发：小店接到一杯饮品订单。推荐动作：`load_roll`、`place_cup`、`align_film`、`feed`、`press_seal`、`release`、`inspect`、`handover`。

物理锁：卷材从卷芯连续送入机器，印刷重复节距和方向一致；杯沿与膜面同心接触；机器动作采用来源可见的推入、压合或自动循环方式；封合后只有杯口范围被切断/固定，卷材其余部分连续。

`press_seal` 必须引用同款或明确兼容机器的安装/动作证据，不能用 `route_default` 猜机器结构。只有卷材图、没有机器和封口结果时，停止成品锚点生成并在 TXT 准确说明证据缺口；不得拿竞品机器补造。

可信闭环示例：订单杯放在操作台 → 店员把杯放入封口位 → 膜带自然送到杯口上方 → 机器完成一次克制的压合循环 → 店员提杯检查杯口 → 把饮品递给顾客。只有结果图或演示支持时才轻微倾斜检查渗漏。

禁忌：明火、火花、熔融岩浆质感、手直接碰加热面、膜卷瞬移、杯盖凭空出现、没有机器却自动热封、无证据“100% 不漏”。

## surface_covering

适用：桌布、桌垫、墙纸、家具保护膜、抽屉垫。`form_factor` 使用 `textile_panel` 或 `roll_to_sheet`。

推荐触发：准备吃饭、做手工、整理桌面或铺设保护层。推荐动作：`clear_surface`、`measure`、`unfold`、`align`、`smooth`、`trim`、`place_items`、`resume_use`。

物理锁：材料沿表面受重力和摩擦铺开；厚度、卷曲记忆、边缘和图案比例稳定；桌角与膜/布的包覆方式采用来源证据。

可信闭环示例：移开桌面物品 → 展开并对齐边缘 → 双手向外抚平 → 检查四角 → 把餐具或作业本放回 → 日常使用继续。

禁忌：表面自动翻新、划痕魔法消失、材质变成镜面、未经证明的耐高温/防刮/抗菌结论。

## seasonal_decor

适用：节庆窗贴、灯笼、挂饰、门贴。按实物选择 `decal_transfer`、`textile_panel` 或 `rigid`。

推荐触发：家人或店员在节日前布置一处普通空间。推荐动作：`choose_position`、`clean`、`peel`、`hang`、`align`、`press`、`secure`、`inspect`、`resume_use`。

物理锁：只出现包装/来源中存在的装饰；悬挂物有真实挂点并受重力；贴纸遵循贴合规则；同一图案、数量和节庆语义保持不变。

可信闭环示例：普通客厅窗边准备布置 → 选一个位置并清洁 → 揭贴/挂上装饰 → 调整水平与边缘 → 家人从正常距离检查 → 回到用餐或店铺营业。

禁忌：烟花、魔法粒子、满屋自动变装、无来源的儿童或大型派对、悬挂物无挂点漂浮。

## fallback_household

适用：无法可靠归入以上路线的家居小商品。`form_factor` 使用 `rigid`、`textile_panel` 或来源可证明的形态。

只采用保守动作：`pick_up`、`prepare`、`align`、`place`、`attach`、`operate`、`inspect`、`resume_use`。从一个普通需求开始，在同一空间完成一次真实使用，再回到日常。看不出安装方式就只做拿取、摆放、开合或外观检查，不发明机关和功效。

## 硬失败清单

出现任一项即重写导演表或重做最早的错误锚点：

- 串 SKU、颜色/图案/文字变化、部件增减或商品重复。
- 布料违反重力，网面没有张力，薄膜变硬板，贴纸浮空，卷材送进方向跳变。
- 手与商品没有真实接触，安装方式与来源不一致，机器动作不可能。
- 场景、窗框、桌面、家具、主光或人物身份无转场漂移。
- 黑棚、底座、漂浮、粒子、光束、烟雾、火花、能量特效或夸张反应。
- 以颜色光效证明隔热、防紫外线、防蚊、防水、隔音、耐久等未被结果证据授权的功能。
- 最后一帧脱离生活任务，变成对称英雄海报、促销页或凭空出现的包装阵列。
